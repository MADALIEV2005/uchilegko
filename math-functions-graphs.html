Функции и графики

Функция — это правило, которое каждому значению переменной x ставит в соответствие одно значение y. Проще говоря, функция описывает зависимость между двумя величинами.

Основные понятия:

  • Область определения — все допустимые значения x.
  • Область значения — все возможные значения y.
  • График функции — это изображение всех точек (x, y), удовлетворяющих функции.

Примеры функций:

  • Линейная функция: y = 2x + 1
  • Квадратичная функция: y = x² - 3x + 2
  • Функция обратной пропорциональности: y = 5 / x (при x ≠ 0)

Графики функций:

Графики помогают наглядно понять, как меняется значение функции при изменении x.

Совет: Рисуйте графики функций на бумаге, чтобы лучше понимать их поведение и особенности.
